# Demo Checklist

Use this list while recording the final demo video.

## Before Recording

- Make sure the emulator or physical phone is visible and readable
- Make sure Firebase Console is open if backend features are being shown
- Make sure audio is clear
- Close unrelated windows and notifications

## Demo Flow

1. Cold-start the app
2. Show registration flow
3. Show logout
4. Show login flow
5. Visit each major screen
6. Demonstrate CRUD actions
7. Show the backend reflecting CRUD changes
8. Show one validation error with a polite message
9. Show one saved preference surviving restart

## Recording Rules

- No slide deck
- No team introductions
- Focus on the app only
- Keep the recording continuous if possible
- Target 10 to 15 minutes
